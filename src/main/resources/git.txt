将项目通过git来进行管理
1.自己创建好基本的项目结构
2.将我们的项目变成一个Git仓库  --> git init
3.因为我们的代码仓库只需要存入我们的代码 .class文件
编译器自身的文件都不需要放入仓库中,而且如果放入
还会经常出现代码冲突,解决方案就是在忽略文件中进行设置.gitignore
4.在设置好忽略文件后我们就可以就可以将我们的代码提交到我们的
本地仓库了,这时候对于我们的文件 我们需要先通过 git add 要添加
的文件的路径 来对我们要放入本地仓库的代码
进行添加操作,接下来 通过 git commit -m "提交信息"  来真的
将我们的代码项目提交到本地仓库,这时候才是真的将代码存在本地仓库了
,我们希望跟别人共同进行代码的管理就需要将我们本地仓库和远程git仓库进行连接
并将代码推送到远程服务器上这样才能跟别人进行共同开发
git remote add origin  远程服务器上的仓库地址
这样连接好之后我们就可以进行项目从本地到服务器的推送了
git push -u origin master

合作开发的步骤
1.项目组长将框架搭好,上传到github
2.邀请组员成为协作者
3.组员git clone 项目的克隆拉取
4.coding 完成 进行代码的上传
5.先push 拉取,将服务器上的最新
